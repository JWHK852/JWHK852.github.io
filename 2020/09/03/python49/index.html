配置Redis

下載

點我

解壓

壓縮包>downloads>解壓redis64-2.8.2101.zip

配置

打開redis64-2.8.2101文件夾>打開redis.windows.conf

  1. 在387行設置密碼requirepass <密碼>
  2. 在455行添加maxheap 1024000000

Redis 詳細配置

  1. 主機ip：127.0.0.1
  2. 端口號：6379
  3. 數據文件：dump.rdb

啟動服務

  1. 命令行進入安裝目錄
  2. 輸入指令redis-server.exe redis.windows.conf
  3. 再打開另一個命令行，進入安裝目錄，輸入指令redis-cli.exe
  4. auth '密碼'表示鍵入密碼，出現OK表示進入
  5. set name 字符串創建 鍵

可視化工具

  • redis desktop manager
    點我下載

連接步驟

數據操作

  • Redis是key_value的數據，所以每一個數據都是一個鍵值對
  • 鍵的類型是字符串
  • 值的類型分為五種
    1. 字符串string
    2. 哈希hash
    3. 列表list
    4. 集合set
    5. 有序集合zset
  • 數據操作的全部指令
    • 中文版
    • 英文版

數據類型

String

  • 概述：String是Redis最基本的類型，最大能存儲512MB的數據，String類型是二進制安全的，及可以存儲任何數據比如數字，圖片，序列化對象等。（只要能變二進制的都能存）

設置

  1. 設置鍵值

    • set key value
    • 示例：set a jerry

  2. 設置鍵值以及過期時間，以秒為單位

    • setex key seconds value
    • 示例（十秒後c鍵消失）：setex c 10 good

  3. 設置多個鍵值

    • mset key value key value key value ...
    • 示例存入3條鍵值對：mset d good e nice f wow

獲取

  1. 根據鍵獲取值，如果鍵不存在則返回None(null或0或nil)

    • get key
    • 示例：get d

  2. 根據多個鍵獲取多個值

    • mget key key...
    • 示例：mget d e

運算

  • 要求：值是字符串類型的數字
  1. 將key對應的值加1
    • incr key
    • 示例：incr d
  2. 將key對應的值減1
    • decr key
    • 示例：decr d
  3. 將key對應的值加整數
    • incrby key intnum
    • 示例（d的值加9）：incrby d 9
  4. 將key對應的值減整數
    • decrby key intnum
    • 示例（d的值減9）：decrby d 9

其他

  1. 追加值
    • append key value
    • 示例（在b的值後追加一個!）：append b !
  2. 獲取長度
    • strlen key
    • 示例（獲取b的長度）：strlen b

Key

  1. 查找鍵，參數支持正則表達式
    • keys pattern
    • 示例（查看所有鍵）:keys *
  2. 判斷鍵是否存在，存在返回1，不存在返回0
    • exists key
    • 示例（查看b鍵是否存在）：exists b
  3. 查看鍵對應的value類型
    • type key
      • 示例（查看g鍵的類型）：type g
  4. 刪除鍵及對應的值
    • del key key...
    • 示例（刪除f和g鍵）：del f g
  5. 設置過期時間，以秒為單位
    • expire key seconds
    • 示例（a鍵五秒後過期）：expire a 5
  6. 查看有效時間，以秒為單位
    • ttl key
    • 示例（查看a鍵多少秒後過期）：ttl a

  • 例子：

設置

  1. 設置單個值

    • hset key field value
    • 示例：hset p1 name Jerry

  2. 設置多個值

    • mhset key field value field value...
    • 示例：hmset p2 name Irene age 17

獲取

  1. 獲取一個屬性的值
    • hget key field
    • hget p2 name
  2. 獲取多個屬性的值
    • hmget key field field ...
    • 示例：hmget p2 name age
  3. 獲取所有屬性和值
    • hgetall key
    • 示例：hgetall p2
  4. 獲取所以屬性
    • hkeys key
    • 示例：hkeys p2
  5. 獲取所有值
    • hvals key
    • 示例：hvals p2
  6. 返回包含數據的個數
    • hlen p1

其他

  1. 判斷屬性是否存在，存在返回1，不存在返回0
    • hexists key field
    • 示例：hexists p2 name
  2. 刪除屬性及值
    • hdel key field field...
    • hdel p1 name age
  3. 返回值的字符串長度
    • hstrlen key field

list

  • 概述：列表的元素類型為String，按照插入順序排序，在列表的頭部或尾部添加元素

設置

  1. 在頭部插入
    • lpush key value value...
    • 示例：lpush s1 1
  2. 在尾部插入
    • rpush key value value...
    • 示例：rpush s1 2
  3. 在一個元素的前或後插入新元素
    • linsert key before|after pivot value
    • 示例：linsert s1 after 1 4
  4. 設置指定索引的元素值
    • lset key index value
    • 注意：index從0開始。索引值可以是負數，表示偏移量是從list的尾部開始，如-1表示最後一個元素
    • 示例：lset s1 2 40

獲取

  1. 移除並返回key對應的list的第一個元素
    • lpop key
    • 示例：lpop s1
  2. 移除並返回key對應的list的最後一個元素
    • rpop key
  3. 返回存儲在key的列表中的指定範圍的元素
    • lrange key start end
    • 示例：lrange s1 0 2
    • 注意：strat end 都是從0開始，偏移量可以是負數

其他

  1. 裁剪列表，改為原集合的一個子集
    • ltrim key start end
    • 示例：ltrim s1 0 2
    • 注意：strat end 都是從0開始，偏移量可以是負數
  2. 返回儲存在key裡的list的長度
    • llen key
    • llen s1
  3. 返回列表中索引對應的值
    • lindex key index
    • lindex s1 1

Set

  • 概述：無序集合，元素類型為String類型，而且元素具有唯一性（不重複）

設置

  1. 添加元素
    • sadd key member member...

獲取

  1. 返回key集合中所有元素
    • smembers key
    • 示例：smembers d1
  2. 返回集合元素個數
    • scard key
    • 示例：scard d1

其他

  1. 求多個集合的交集
    • sinter key key...
    • 示例：sinter d1 d2
  2. 求多個集合的差集
    • sdiff key key...
    • 示例：sdiff d1 d2
  3. 求多個集合的合集
    • sunion key key...
    • 示例：sunion d1 d2
  4. 判斷元素是否在集合中，存在返回1，不存在返回0
    • sismember key member
    • 示例：sismember d1 2

Zset

  • 概述：有序集合，元素類型為String，而且元素具有唯一性，不能重複
  • 每個元素都會關聯一個double類型的score（表示權重），通過score的大小排序，元素的score可以相同

設置

  1. 添加
    • zadd key score member score member...
    • 示例：zadd z1 1 a 5 b 3 c 2 d 4 e

獲取

  1. 返回指定範圍內的元素
    • zrange key start end
    • 示例：zrange z1 0 -1
    • 注意：strat end 都是從0開始，偏移量可以是負數
  2. 返回元素個數
    • zcard key
    • 示例：zcard z1
  3. 返回有序集合key中，score在min和max之間的元素的個數
    • zcount key min max
    • 示例：zcount key 2 4
  4. 返回有序集合key中，成員member的score值
    • zscore key member
    • 示例：zscore z1 c

Redis 用途
