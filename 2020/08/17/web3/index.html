Web3-CSS

發表於2020-08-17|更新於2020-08-22|計算機語言
|閱讀量:

簡介

  • CSS：層疊樣式表
  • 網頁是一個多層的結構，通過CSS可以分別為網頁的每一層進行設置樣式，而最終我們能看到的只是網頁的最上面一層
  • 總之 CSS用來設置網頁中的元素樣式

使用CSS修改元素的樣式

第一種方式：內聯樣式

  • 在標籤內部通過style屬性設置元素的樣式
  • 開發中不推薦使用style屬性
    • 問題
      • 使用內聯樣式只能對一個標籤生效，如果要影響多個標籤就需要輸入多個內聯樣式
      • 並且當樣式發生變化時我們要一個個修改，非常不方便維護
效果

I switch up my cup I kill any pain

第二種方式：內部樣式表

  • 將樣式編寫到head中的style標籤裡，然後通過CSS選擇器選中元素並為其設置各種樣式
  • 好處：可以同時為多個標籤設置樣式，並且修改時只需要修改一處即可全部應用
  • 問題：內部樣式表不能跨頁面進行複用
效果

I'd do anything in my power

to see you just smile

第三種方式：外部樣式表

  • 可以將css樣式編寫到一個外部的css文件中，然後通過link標籤引入外部的css文件
  • 步驟
    1. 在同目錄下新建style.css
    1. 引用並顯示style.css的效果

    效果

    So call out my name

    Call out my name when I kiss you so gently

  • 外部樣式表需要通過link標籤進行引入，意味著只要使用這些樣式的網頁都可以對其進行引用，使樣式可以在不同的頁面進行複用
  • 將樣式編寫到外部的CSS文件中，可以使用到瀏覽器的緩存機制，從而加快網頁的加載速度，提高用戶體驗

CSS語法

注释

選擇器與聲明塊

  • 選擇器

    • 通過選擇器可以選中頁面中的指定元素
      • 比如p的作用就是選中頁面中的所有的p元素

  • 聲明塊

    • 通過聲明塊來指定要為元素設置的樣式
    • 聲明塊由一個個聲明組成
    • 聲明是一個名值對結構
      • 一個樣式名對應一個樣式值，名和值之間用:連接，以;結尾

常用選擇器

元素選擇器

  • 可以根據標籤名選中指定元素
  • 語法：標籤名{}
    • 例子
      • p{}
      • h1{}
      • div{}

ID選擇器

  • 根據元素的id屬性值選中一個元素
  • 語法：#id屬性值{}
    • 例子：
      • #box{} #red{}

實例

  • 將你好變紅變大，Hello保持不變
  • 效果:

    你好

    Hello

類選擇器

  • 作用：根據元素的class屬性選中一組元素
  • 語法：.class屬性值
  • 可以同時為一個元素指定多個class屬性，多class用空格來分隔

實例

  • 將你好,wow變紅。Hello,Nyoooom變藍
  • 效果：

    你好

    Hello

    wow

    Nyoooom

通配選擇器

  • 作用：選中頁面中所有的元素
  • 語法：*{}

交集選擇器

  • 作用：選中同時符合多個條件的元素
  • 語法：選擇器1選擇器2選擇器3...選擇器n{}

實例

  • 將class為red的div字體顏色改為紅色
  • 效果：
    我是div

    我是p元素

並集選擇器

  • 作用：同時選中多個選擇器對應的元素
  • 語法：選擇器1,選擇器2,選擇器3,...,選擇器n{}
  • 將div和span元素的字體顏色改為紅色
  • 效果：

    我是span

    我是div

    我是p元素

關係選擇器

關係說明

  • 父元素

    • 直接包含子元素的元素叫父元素

  • 子元素

    • 直接被父元素包含的元素叫子元素

  • 祖先元素

    • 直接或間接包含後代元素的元素叫祖先元素
    • 一個元素的父元素也是它的祖先元素

  • 後代元素

    • 直接或間接被祖先元素包含的元素叫後代元素
    • 子元素也是後代元素

  • 兄弟元素

    • 擁有相同父元素的元素是兄弟元素

  • 舉例說明

    • div元素是p元素的父元素
    • p元素是div元素的子元素
    • div元素是p元素，span元素與q元素的祖先元素
    • p元素，span元素與q元素是div元素的後代元素
    • p元素與span元素是兄弟元素

子元素選擇器

  • 作用：選定父元素的指定子元素
  • 語法：父元素 > 子元素{}
  • 實例
    • 為div的子元素span設置一個字體顏色：orange

後代元素選擇器

  • 作用：選定祖先元素的指定後代元素
  • 語法：祖先元素 後代元素{}
  • 實例
    • 為div的後代元素q設置一個字體顏色：skyblue

兄弟元素選擇器

  • 選擇下一個兄弟元素語法：前一個 + 後一個{}
  • 選擇下面所有的兄弟元素語法：兄 ~ 弟{}
  • 實例
    • 為p的span設置一個字體顏色：green
  • 效果

屬性選擇器

  • 作用：
    • 選擇含有指定屬性的元素語法：[屬性名]{}
    • 選擇含有指定屬性和屬性值的元素語法：[屬性名=屬性值]{}
    • 選擇屬性值以指定值開頭的元素語法：[屬性名^=屬性值]{}
    • 選擇屬性值以指定值結尾的元素語法：[屬性名$=屬性值]{}
    • 選擇屬性值含有某值的元素語法：[屬性名*=屬性值]{}
  • 實例
    • 為有title的元素設置一個字體顏色：orange

偽類選擇器

  • 偽類：不存在的類
    • 用來描述一個元素的特殊狀態
    • 比如：第一個元素，被點擊的元素，鼠標移入的元素
  • 偽類一般情況都是用:開頭
    • :first-child:第一個子元素
    • :last-child:最後一個子元素
    • :nth-child(n):選中第n個子元素
      • 特殊值
        • n：第n個，n的範圍0到+∞
        • 2n 或 even：選中偶數位的元素
        • 2n+1 或 odd：選中奇數位的元素
      • 以上的偽類都是根據所有的子元素進行排序
    • :first-of-type
    • :last-of-type
    • :nth-of-type
      • 以上三個偽類的功能與上述的類似，不過他們是與同類型的元素進行排序
    • :not():否定偽類
      • 將符合條件的元素從選擇器中去處

    • dd

    • 第0個
    • 第一個
    • 第二個
    • 第三個
    • 第四個
    • 第五個

  • 修改訪問過的鏈接的顏色
    • :link：用來表示未訪問過的鏈接（正常的鏈接）

樣式的繼承

  • 為一個元素設置的樣式會同時被他的後代元素應用
  • 繼承的設計是為了方便我們的開發，利用繼承可以將通用的樣式統一設置到共同的祖先元素上，這樣只需設置一次所有元素便獲取該樣式
  • 注意：並不是所有樣式都會被繼承，比如：
    • 背景相關
    • 佈局相關

我是p元素 我是p元素中的span

選擇器的權重

  • 樣式衝突
    • 通過不同的選擇器選中相同的元素，並且為相同的樣式設置不同的值，此時就發生了樣式的衝突
    • 發生樣式衝突是應用哪個樣式由選擇器的權重決定
  • 選擇器的權重排名
    1. 內聯樣式（優先級：1,0,0,0）
    2. id選擇器（優先級：0,1,0,0）
    3. 類和偽類選擇器（優先級：0,0,1,0）
    4. 元素選擇器（優先級：0,0,0,1）
    5. 通配選擇器（優先級：0,0,0,0）
    6. 繼承的樣式（沒有優先級）
  • 比較優先級時，需要將所有選擇器的優先級進行相加計算，最後優先級越高，則優先顯示（分組選擇器是單獨計算）
    • 選擇器的累加不會超過其最大的數量級，類選擇器再高也不會超過id選擇器
    • 如果優先級計算後相等，此時則優先使用靠下的樣式
  • 可以在某一個樣式後邊添加!important，則會獲得最高的優先級，優先級超過內聯樣式。
    • 注意：在開發中!important要慎用

單位

  • 長度單位
    • 像素
    • 百分比
      • 也可以將屬性值設置為相對於其父元素屬性的百分比
      • 設置百分比可以使子元素跟隨父元素的改變而改變
    • em
      • em是相對與元素的字體大小計算的
        • 1em = 1font-size
        • em 會根據字體大小改變而改變
    • rem
      • rem是相對於根元素字體大小計算的

顏色

  • 顏色單位
    • RGB
      • RGB通過三種不同顏色的濃度來調配不同的顏色
      • R = Red, G = Green, B = Blue
      • 每一種顏色的範圍在0~255之間
    • RGBA
      • RGBA通過四種不同顏色的濃度來調配不同的顏色
      • A為不透明度
      • R = Red, G = Green, B = Blue, A = Alpha
    • 十六進制RGB
      • 語法：#紅色綠色藍色
      • 顏色濃度通過00~ff表示
      • 如果顏色兩位重複可以進行簡寫
        • #aabbcc --> #abc
    • HSL值
      • H = 色相(0~360)，S = 飽和度(0%~100%)，L = 量度(0%~100%)
    • HSLA值
      • H = 色相，S = 飽和度，L = 量度，A = 不透明度
