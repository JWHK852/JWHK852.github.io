HTML

  • HTML(HyperText Markup Language)超文本標記語言
  • 它負責網頁的三個要素之中的結構
  • HTML使用標籤的形式來標識網頁中的不同組成部分
  • 所謂的超文本指的是超鏈接，使用超鏈接可以讓我們從一個頁面跳轉到另一個頁面

編寫第一個網頁

  • <!DOCTYPE html>聲明為HTML5文檔
  • <html>元素是HTML頁面的根元素
  • <head>元素包含了文檔的元（meta）數據，如<meta charset =“ utf-8”>定義網頁編碼格式為utf-8。
  • <title>元素描述了文檔的標題
  • <body>元素包含了可見的頁面內容
  • <h1>元素定義一個大標題
  • <p>元素定義一個起點

屬性

注釋

文檔聲明<!doctype html>

編碼

  • 使用<meta charset =“ utf-8”>編碼

實體

  • 在 HTML 中，某些字符是預留的。
  • 在 HTML 中不能使用小於號（<）和大於號（>），這是因爲瀏覽器會誤認爲它們是標籤。
  • 如果希望正確地顯示預留字符，我們必須在 HTML 源代碼中使用字符實體（character entities）。

meta標籤

  • 主要用於設置網頁中的元數據，供瀏覽器或搜索引擎看

屬性

  • charset：固定網頁字符集
  • name：指定的是數據的名稱
  • content：指定的是數據的內容
    • keywords：表示網站的關鍵字，可以使用多個關鍵字，關鍵字使用,隔開
    • description：用於指定網站的描述，網站的描述會顯示在搜索引擎的搜索結果中
  • http-equiv：網頁跳轉

塊元素

  • 在網頁中一般通過塊元素來對頁面佈局

標題標籤

  • 一共有六級標題h1-h6
  • 從h1-h6重要性遞減，h1最重要，h6最不重要
  • h1在網頁中的重要性僅次於title標籤，一般情況下頁面只會有一個h1
  • 一般情況下只會使用到h1-h3，h4-h6很少用
  • 在頁面中獨佔一行的元素稱之為塊元素（block element）

標題組

  • hgroup標籤用來為標籤分組，可以將一組相關的標題同時放入到hgroup

段落標籤

  • p標籤表示頁面中的一個段落
  • p標籤也是塊元素

長引用

  • 用來引用較長的內容

行內元素

  • 在頁面中不會獨佔一行的元素成為行內元素（inline element）
  • 行內元素主要用來包裹文字
  • 一般情況下會在塊元素內放行內元素，而不會在行內元素內放塊元素
  • p元素中不能放任何的塊元素

em標籤

  • 字體會斜體

strong標籤

  • strong標籤表示強調重要內容（粗體）

短引用

  • 用來引用較短的內容

br標籤

  • 換行

佈局標籤（結構化語義標籤）

header，main，footer

  • header：表示網頁的頭部
  • main：表示網頁的主體（一個頁面中只有一個main）
  • footer：表示網頁的底部
  • 示意圖

nav

  • 表示網頁中的導航

aside

  • 表示和主體相關的其他內容

article

  • 表示一個獨立的文章

section

  • 表示一個獨立的區塊，上面的標籤都不能表示時，可以使用section

div

  • 沒有語義，用來表示一個區塊，目前來說div使我們主要的佈局元素

span

  • span是一個行內元素，沒有任何語義，一般用於網頁中選中文字

列表

  • HTML中的列表有三種
    1. 有序列表
    2. 無序列表
    3. 定義列表
  • 列表之間可以互相嵌套

有序列表

  • 使用ol標籤創建無序列表
    • 使用li表示列表項

無序列表

  • 使用ul標籤創建無序列表
    • 使用li表示列表項

定義列表

  • 使用dl標籤創建定義列表
    • 使用dt來表示定義的內容
    • 使用dd來對內容進行解釋說明

超鏈接

  • 超鏈接可以讓我們從一個頁面跳轉到其他頁面，或者是我們當前頁面的其他的指定位置
  • 使用a標籤來定義超鏈接
  • 超鏈接a標籤是行內元素，在a標籤中可以嵌套除了它自身外的任何的元素

屬性

id

  • id屬性是唯一不重複的
  • 每一個都可以添加一個id屬性，是元素唯一標識，同一個頁面中不能出現同一個id屬性

href

佔位

  • 可以使用javascript:;作為屬性的佔位符，點擊不會發生任何事情

跳轉外部頁面

  • href 指定跳轉的目標路徑
    • 值可以是一個外部網站的地址
    • 也可以寫一個內部頁面的地址

跳轉內部頁面

  • 當我們需要跳轉到一個服務器內部的頁面時，一般我們會使用相對路徑。
    • 相對路徑一般都會使用./或者../開頭
      • ./ 表示當前文件所在的目錄
      • ../表示當前文件所在的上層目錄

跳轉頁面指定位置

  • 可以直接把超鏈接的href屬性設置為#，點擊會轉到當前頁面的頂部
  • 可以跳轉到頁面的指定位置，只需將href屬性設置為#目標元素的id屬性值

target

  • target屬性，用來指定超鏈接打開的位置
    • 可選值
      • _self：默認值，在當前頁面中打開超鏈接
      • _blank：在一個新的標籤打開超鏈接

圖片

圖片格式

  1. JPRG
    • 支持的顏色豐富，不支持透明效果，不支持動圖
    • 一般用來顯示照片
  2. PNG
    • 支持的顏色豐富，支持複雜的透明效果，不支持動圖
  3. GIF
    • 支持的顏色較少，支持簡單的透明效果，支持動圖
  4. WEBP
    • Google推出的專門用來表示網頁中的圖片的格式
    • 它具備其他圖片格式的所有優點，而且文件還特別小
    • 缺點：兼容性不好
  5. base64
    • 將圖片使用base64進行編碼，這樣可以將圖片轉換為字符，通過字符的形式來引入圖片
    • 一般都是一些需要和網頁一起加載的圖片才會使用base64

圖片標籤

  • 用於向當前頁面中引入外部圖片
  • 使用img標籤引入外部圖片。img標籤是一個自結束標籤
  • img屬於替換元素（塊和行內元素之間，具有兩種元素的特點）
  • 屬性
    • src：屬性指的是外部的圖片路徑
    • alt：圖片描述，默認情況下不會顯示，有些瀏覽器會在圖片無法加載時顯示。搜索引擎會根據alt中的內容識別圖片，如果不寫alt屬性則圖片不會被搜索引擎收錄
    • 長寬
      • width：圖片的寬度（單位是像素）
      • height：圖片的高度（單位是像素）
      • 高度或寬度中如果只修改了一個，則另一個會被等比縮放
      • 注意：一般情況下pc端不建議修改圖片大小，需要多大圖片就裁多大。但是在移動端經常需要對圖片進行縮放（主要大圖縮小）

效果圖

內聯框架

  • 用於向當前頁面中引入一個其他頁面
  • 屬性
    • scr：要引入的網頁的路徑
    • width：網頁寬度
    • height：網頁高度
    • frameborder：指定內聯框架的邊框

音視頻

  • 音視頻文件引入時默認情況下不允許用戶自己控制播放或停止

音頻標籤

  • audio標籤用來向頁面引入一個外部的文件
  • 屬性
    • controls：是否允許用戶控制播放
    • autoplay：音頻文件是否自動播放
      • 如果設置了autoplay則音樂在打開頁面時會自動播放，但是目前大部分瀏覽器都不會自動對音樂進行播放。
    • loop：是否循環播放
    • 除了通過src來指定外部文件的路徑外，還可以用source指定文件路徑
      • 特點：
        • 可以同時指定多個文件
        • 可以為不兼容的瀏覽器顯示提示
        IE8:

embed

  • 使用自帶的media player播放音視頻，兼容老版本瀏覽器
  • 屬性
    • width：播放器寬度
    • type：指定文件類型
    • src：文件源地址

新版本用audio標籤，老版本用embed標籤解決方法

影片標籤

  • 使用video標籤來向網頁引入一個影片
  • 屬性
    • controls：是否允許用戶控制播放
    • autoplay：影片文件是否自動播放
      • 如果設置了autoplay則影片在打開頁面時會自動播放，但是目前大部分瀏覽器都不會自動對影片進行播放。
    • loop：是否循環播放
    • 除了通過src來指定外部文件的路徑外，還可以用source指定文件路徑
      • 特點：
        • 可以同時指定多個文件
        • 可以為不兼容的瀏覽器顯示提示
        IE8:

新版本用audio標籤，老版本用embed標籤解決方法

class

  • class是一個標籤屬性，它和id相似，不同的是class可以重複使用

快捷鍵

快速創建父子類

  • 格式：父類元素>子類元素，然後按Tab

創建多個子類

  • 格式：父類元素>子類元素*數量，然後按Tab

快速注釋

  • ctrl + /
