Python16-函數

發表於2020-08-06|更新於2020-08-16|計算機語言

函數簡介

  • 在一個完整的項目中，有某些功能會反復地使用。那麼會將功能封裝成函數，當我們要使用功能的時候，直接調用函數即可。

本質

  • 函數就是對功能的封裝

優點

  1. 簡化代碼結構，增加了代碼的複用度
  2. 如果想修改某些功能或者調試某個BUG，只需要修改對應的函數即可

定義函數

格式

  • 解釋

    • def：函數代碼塊以關鍵字"def"開始
    • 函數名：遵循標識符規則
    • ()：是參數列表的開始和結束
    • 參數列表（參數1, 參數2, …, 參數n）：任何傳入函數的參數和變量必須放在"()“之間，用”,"分隔。函數從函數的調用者（調用的程序）那裡獲取信息。
    • 冒號":"：函數內容（封裝的功能）以":"開始，並且縮進
    • 語句：函數封裝的功能
    • return：一般用於結束函數，並返回信息給函數的調用者（調用的程序）
    • 表達式：即為要返回給函數的調用者（調用的程序）的信息

  • 注意：最後的"return 表達式" 可以不寫，相當於"return None"

調用函數

格式

  • 解釋
    • 函數名：是要使用的功能的函數名字
    • 參數列表：函數的調用者給函數的傳遞的信息，如果沒有參數，"()"也不能省略

傳遞參數

值傳遞

  • 傳遞的是不可變類型
    1. string
    2. tuple
    3. number
  • 不會影響外面的值

引用傳遞

  • 傳遞的是可變類型
    1. list
    2. dict
    3. set
  • 會影響外面的值

關鍵字參數

  • 允許函數調用時參數順序與定義時不一致

默認參數

  • 調用函數時，如果沒有傳遞參數，則使用默認參數
  • 要用默認參數，最好將默認參數放到最後
