Json

發表於2020-08-25|更新於2020-08-25|計算機語言
|閱讀量:

簡介

  • 概念：Json是一種保存數據的格式
  • 作用：可以保存本地的json文件，也可以將json串進行傳輸，通常將json成為輕量級的傳輸方式

Json的組成

  • {}：代表對象（字典）
  • []：代表列表
  • :：代表鍵值對
  • ,：分隔兩個部分

Json與Python數據類型對象互相轉換

  • json轉dict: json.loads()
  • dict轉json: json.dumps()

讀取本地json文件

  • 讀取後自動轉為dict文件

寫入本地json文件

path2 = r"C:\Users\Jerry\Desktop\code\pycharm\qianfeng\爬虫\file\file2.json"
jsonData3 = {"name": "Jerry", "age": 17, "hobby": ["gaming", "coding"], "parames": {"a": 1, "b": 2}}
with open(path2, "w") as f:
    json.dump(jsonData3, f)```
