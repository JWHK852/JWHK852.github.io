Python42-正則表達式

發表於2020-08-23|更新於2020-08-26|計算機語言
|閱讀量:

概述

  • Python1.5以後增加了re的模塊，提供了正則表達式模式
  • 貪婪匹配：盡可能多的匹配

re模塊

  • re模塊使Python語言擁有了全部的正則表達式功能

re.match()

  • re.match函數
    • 原型：match(pattern, string, flags=0)
    • 參數：
      • pattern：匹配的正則表達式
      • string：要匹配的字符串
      • flags：標誌位，用於控制正則表達式的匹配方式，值如下
        1. re.I：忽略大小寫
        2. re.L：做本地化識別
        3. re.M：多行匹配，影響^和$
        4. re.S：使.匹配包括換行符在內的所有字符
        5. re.U：根據Unicode字符集解析字符，影響\w \W \b \B
        6. re.X：使我們以更靈活的格式理解正則表達式
    • 功能：嘗試從字符串的起始位置匹配一個模式，匹配不成功的話，會返回None。如果不是起始位置匹配成功的話，也會返回None。

正則表達式的元字符

  1. .：匹配除換行符以外的任意字符
  2. []：字符集合，表示匹配方括號中所包含的任意一個字符
    1. [a-z]：匹配任意小寫字母
    2. [A-Z]：匹配任意大寫字母
    3. [0-9]：匹配任意數字
    4. [0-9a-zA-Z]：匹配任意的數字和字母
    5. [0-9a-zA-Z_]：匹配任意的數字和字母和下劃線
    6. [^abc]：匹配除了abc這幾個字母以外的所有字符，中括號裡的^稱為脫字符，表示不匹配集合中的字符
    7. [^0-9]：匹配所有的非數字字符
  3. \d：匹配數字，效果同[0-9]
  4. \D：匹配非數字，效果同[^0-9]
  5. \w：匹配數字，字母和下劃線，效果同[0-9a-zA-Z_]
  6. \W：匹配非數字，字母和下劃線，效果同[^0-9a-zA-Z_]
  7. \s：匹配任意的空白符（空格，換行，回車，換頁，製表）效果同[ \f\n\r\t]
  8. \S：匹配任意的非空白符效果同[^ \f\n\r\t]

示例

錨字符（邊界字符）

  1. ^：行首匹配（和在[]裡的^不是一個意思）
  2. $：行尾匹配
  3. \A：匹配字符串開始，它和^的區別是\A只匹配整個字符串的開頭，即使在re.M模式下，它也不會匹配它行的行首
  4. \Z：匹配字符串結束，它和$的區別是，\Z只匹配整個字符串的結束，即使在re.M模式下，它也不會匹配它行的行尾
  5. \b：匹配一個單詞的邊界，也就是值單詞和空格間的位置
  6. \B：匹配非單詞邊界

示例

匹配多個字符

  • 說明：下方的x, y, z, n, m均為假設的普通字符，不是正則表達式的元字符。
    1. (xyz)：匹配小括號內的xyz（作為一個整體去匹配）
    2. x?：匹配0個或者1個x
    3. x*：匹配0個或者任意多個x（.* 表示匹配0個或者任意多個字符（換行符除外））
    4. x+：匹配至少一個x
    5. x{n}：匹配確定的n個x（n是一個非負整數）
    6. x{n,}：匹配至少n個x
    7. x{n,m}：匹配至少n個最多m個x，注意：必須n≤m
    8. x|y：|表示或，匹配的是x或y

re.finditer()

  • re.finditer()函數
    • 原型：finditer(pattern, string, flags=0)
    • 參數：
      • pattern：匹配的正則表達式
      • string：要匹配的字符串
      • flags：標誌位，用於控制正則表達式的匹配方式
    • 功能：與finall類似，掃描整個字符串，返回的是一個迭代器

字符串的替換和修改

  • 原型：
    1. sub(pattern, repl, string, count=0)
    2. subn(pattern, repl, string, count=0)
  • 參數：
    • pattren：正則表達式（規則）
    • repl：指定的用來替換的字符串
    • string：目標字符串
    • count：最多替換次數
  • 功能：在目標字符串中以正則表達式的規則匹配字符串，再把它們替換成指定的字符串，可以指定替換的次數，如果不指定，他會替換所有的匹配字符串
  • 區別：sub返回一個被替換的字符串，subn返回一個元組，第一個元素是被替換的字符串，第二個元素是表示被替換的次數

分組

  • 概念：除了簡單的判斷是否匹配之外，正則表達式還有提取子串的功能。用()表示的就是提取分組，括號內的?P<>是為組起名

編譯

  • 當我們使用正則表達式時，re模塊會幹兩件事

    1. 編譯正則表達式，如果正則表達式本身不合法，會報錯
    2. 用編譯後的正則表達式去匹配對象

  • 原型：compile(pattern, flags=0)

  • 參數：

    • pattern：要變異的正則表達式
    • flags：（不用多說了吧）

  • 功能：將正則表達式編譯為正則對象

  • 例子：匹配中國電話號碼

對應方法

re模塊調用 re對象調用
re.match(pattern, string, flags=0) 正則對象.match(string)
re.search(pattern, string, flags=0) 正則對象.search(string)
re.findall(pattern, string, flags=0) 正則對象.findall(string)
re.finditer(pattern, string, flags=0) 正則對象.finditer(string)
re.split(pattern, string, maxsplit=0, flags=0) 正則對象.split(string, maxsplit=0)
re.sub(pattern, repl, string, count=0, flags=0) 正則對象.sub(repl, string, count=0)
re.subn(pattern, repl, string, count=0, flags=0) 正則對象.subn(repl, string, count=0)
