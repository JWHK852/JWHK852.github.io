Web4-佈局

發表於2020-08-20|更新於2020-09-13|計算機語言
|閱讀量:

文檔流

  • 文檔流（Normal Flow）
    • 網頁是一個多層的結構，一層疊著一層
    • 通過CSS可以分別為每一層設置樣式
    • 作為用戶來說只能看到最頂一層
    • 這些層中，最底下的一層稱為文檔流，文檔流是網頁的基礎
      • 我們所創建的默認元素都是在文檔流中進行排列
    • 對於我們來說元素主要有兩個狀態
      • 在文檔流中
      • 不在文檔流中（脫離文檔流）
    • 元素在文檔流中有什麼特點
      • 塊元素
        • 塊元素會在頁面獨佔一行
        • 默認寬度是父元素的全部（會把父元素撐滿）
        • 默認高度是被內容撐開（被子元素撐開）
        • 塊元素在頁面中由上到下垂直排列
      • 行內元素
        • 行內元素不會獨佔頁面一行，只佔自身大小
        • 行內元素在頁面中由左到右水平排列
        • 如果一行中容納不下所有的行內元素，則元素會換到第二行繼續由左到右排列
          +行內元素的默認寬度和高度都是被內容撐開

盒模型

  • 盒模型（Box Model）
    • CSS將頁面中的所有元素都設計成了一個矩形盒子
    • 將元素設置為矩形盒子後，對頁面的佈局就變成了將不同的盒子擺放到不同的位置
    • 每一個盒子都有以下部分組成

內容區(Content)

  • 元素中的所有的子元素和文本都在內容區中排列。
  • 內容區的大小由width與height兩個屬性設置

邊框（Border）

  • 邊框屬於盒子的邊緣，邊框裡屬於盒子的內部，邊框外屬於盒子的外部
  • 邊框大小會影響整個盒子的大小
  • 要設置邊框至少需要三個樣式

邊框寬度

  • 邊框寬度 border-width
  • 值的情況
    • 四個值：上 右 下 左
    • 三個值：上 左右 下
    • 兩個值：上下 左右
    • 一個值：上下左右
  • 除了border-width還有一組 border-xxx-width,用來單獨指定某個邊的寬度
    • xxx可以是top，right，bottom，left

邊框顏色

  • 邊框顏色 border-color
  • 除了border-color還有一組 border-xxx-color,用來單獨指定某個邊的顏色
    • xxx可以是top，right，bottom，left
  • 規則和border-width一樣
  • border-color也可以省略不寫，如果不寫則顯示color的顏色值

邊框樣式

  • 邊框樣式 border-style
  • solid：實線，dotted：點狀虛線，dashed：虛線，double：雙線
  • 除了border-style還有一組 border-xxx-style,用來單獨指定某個邊的風格
    • xxx可以是top，right，bottom，left
  • 規則和border-width一樣
  • border-style默認屬性為none

簡寫屬性

  • border簡寫屬性，通過該屬性可以同時設置邊框所有相關樣式，並且沒有順序要求
  • 例子
    • border: solid 10px orange
  • 除了border以外還有四個 border-xxx
    • border-top
    • border-right
    • border-bottom
    • border-left

內邊距（Padding）

  • 內容區和邊框之間的距離是內邊距
  • 一共有四個方向的內邊距
    • padding-top
    • padding-right
    • padding-bottom
    • padding-left
  • 內邊距的設置會影響到盒子的大小
  • 背景顏色會延伸到內邊距上
  • 一個盒子的可見框大小由內容區，內邊距和邊框共同決定，所以在計算盒子大小時，需要將這三個區域加到一起計算
  • padding是內邊距的簡寫屬性，可以同時指定四個方向的內邊距，規則和border-width一樣

外邊距（Margin）

  • 外邊距不會影響盒子可見框的大小
  • 但是外邊距會影響盒子的位置
  • 一共有四個方向的外邊距
    • margin-top
      • 上外邊距，設置一個正直，元素會往下移動
    • margin-left
      • 下外邊距，設置一個正直，其他元素會往下移動
    • margin-bottom
      • 下外邊距，設置一個正直，其下面的元素會往下移動
    • margin-right
      • 右外邊距，設置一個正直，其右邊的元素會往右移動
    • margin可以設置負值，如果是負值則元素會向相反的方向移動
    • margin的簡寫屬性
      • margin 可以同時設置四個方向的外邊距，與padding的用法一樣
    • margin會影響對盒子實際佔用空間
  • 元素在頁面中是按照由左向右排列，所以默認情況下如果我們設置的是左和上外邊距則會移動元素自身。而設置下和右會移動其他元素

水平佈局

  • 元素的水平方向的佈局
    • 元素在其父屬性中水平方向的位置有以下幾個屬性共同決定
      • margin-left
      • border-left
      • padding-left
      • width
      • padding-right
      • border-right
      • margin-right
    • 一個元素在其父元素中，他的水平佈局必須滿足以下的等式
      • margin-left + border-left + padding-left + width + padding-right + border-right + margin-right = 其父元素內容區的寬度（必須滿足）
      • 例子：
      • 計算：0 + 0 + 0 + 200 + 0 + 0 + 0 != 800
      • 以上等式必須滿足，如果相加結果使等式不成立，稱為過度約束，則等式會自動調整
      • 調整為：0 + 0 + 0 + 200 + 0 + 0 + 600 = 800
      • 調整情況：
        • 如果這七個值中沒有為auto的情況，則瀏覽器自動調整margin-right值已使等式滿足
    • 這七個值中有三個值可以設置為auto:
      • width, margin-left, margin-right
      • 如果某個值為auto，則會自動調整為auto的那個值以使等式成立
    • 如果將一個寬度和一個外邊距設置為auto，則寬度會調整為最大，設置為auto的外邊距會自動為0
    • 如果三個值都設置為auto，則外邊距都是0，寬度最大
    • 如果將兩個外邊距設置為auto，寬度固定值，則會將外邊距設置為相同的值

垂直外邊距的重疊

  • 相鄰的垂直方向外邊距會發現重疊現象

兄弟元素

  • 兄弟元素間的相鄰垂直外邊距會取兩者之間的較大值（兩者都是正值）
  • 兄弟元素之間的外邊距重疊對於開發是有利的，所以不需要進行處理
我的margin-bottom是100px
我的margin-top是50px
  • 特殊情況
    • 如果相鄰的外邊距一正一負，則取兩者的和
    • 如果相鄰的外邊距都是負值，則取兩者中絕對值較大的

父子元素

  • 父子元素間的相鄰外邊距，子元素的會傳遞給父元素（上外邊距）
  • 父子元素之間的外邊距折疊會影響頁面中的佈局，必須要進行處理

行內元素的盒模型

  • 行內元素不支持設置寬度width和高度height
  • 行內元素可以設置padding，但是垂直方向的padding不會影響頁面的佈局
  • 行內元素可以設置border，垂直方向的border不會影響頁面的佈局
  • 行內元素可以設置margin，垂直方向的margin不會影響頁面的佈局

display屬性

  • 用來設置元素顯示的類型
  • 可選值
    • inline 將元素設置為行內元素
    • block 將元素設置為塊元素
    • inline-block 行內塊元素
      • 行內塊：既可以設置寬度和高度，又不會獨佔一行
    • table 將元素設置成表格
    • none 元素不在頁面中顯示

visibility

  • 可選值
    • visible 默認值，元素在頁面中正常顯示
    • hidden 元素在頁面中隱藏不顯示

默認樣式

  • 通常瀏覽器會為元素設置一些默認樣式
  • 默認樣式的存在會影響到頁面的佈局，通常情況下，編寫網頁時必須去除瀏覽器的默認樣式（PC端的頁面）
  • 重置樣式表：專門用來對瀏覽器的樣式進行
    • reset.css：直接取出瀏覽器的默認樣式
    • normalize.css：對默認樣式進行統一
  • 樣式表1：reset.css
  • 樣式表2：normalize.css

盒子的大小

  • 默認情況下，盒子可見框的大小由內容區、內邊距和邊框共同決定
  • box-sizing：用來設置盒子尺寸的計算方式（設置width和height的作用）
    • 可選值
      • content-box：默認值，寬度和高度用來設置內容區的大小
      • border-box：可選值，寬度和高度用來設置整個盒子的可見框大小
        • width 和 height 指的是內容區和內邊距和邊框的總大小

輪廓，陰影和圓角

輪廓

  • outline：設置元素的輪廓線，用法和border一樣
    • 輪廓和邊框不同點就是輪廓不會影響到可見框的大小

陰影

  • box-shadow：用來設置元素的陰影效果，陰影不會影響頁面佈局
  • 示例：box-shadow: 10px 10px 10px red;
    • 第一個值：水平偏移量，設置陰影的水平位置，正值向右移動，負值向左移動
    • 第二個值：垂直偏移量，設置陰影的垂直位置，正值向下移動，負值向上移動
    • 第三個值：陰影的模糊半徑
    • 第四個值：陰影顏色

圓角

  • border-radius：用來設置圓角，圓角設置圓的半徑大小
  • border-top-left-radius：左上角
  • border-top-right-radius：右上角
  • border-bottom-left-radius：左下角
  • border-bottom-right-radius：右下角

  • border-radius可以分別制定四個角的圓角

    • 四個值 左上 右上 右下 左下
    • 三個值 左上 右上/左下 右下
    • 兩個值 左上/右下 右上/左下
    • 一個值 左上/右下/右上/左下

  • 設置為橢圓border-radius: 20px / 40px

  • 將元素設置為圓形
    • border-radius: 50%;
