Python7-if,if-else,elif語句

發表於2020-07-30|更新於2020-08-09|計算機語言
|閱讀量:

if語句

格式：

邏輯：當程序執行到if語句時，首先計算“表達式”的值。
如果“表達式”的值為真，就執行if下的“語句”。
如果“表達式”的值為假，則跳過整個if語句，繼續向下執行。

何為真假？

假：

  • 0
  • 0.0
  • ‘’ # 空字符串
  • None
  • False

真：除了以上都是真

if-else語句

格式：

邏輯

  • 當程序執行到if-else語句時，首先計算“表達式”的值。
  • 如果“表達式”的值為真，則執行“語句1”。執行完“語句1”跳出整個if-else語句。
  • 如果“表達式”的值為假，則執行“語句2”。執行完“語句2”跳出整個if-else語句。

A[計算表達式的值] 
A --> C[表達式的值為真]
A --> D[表達式的值為假]
D --> E[執行語句2]
C --> F[執行語句1]
F --> G[跳出整個if-else語句]
E --> G

if-elif-else語句

格式

邏輯

當執行if-elif-else語句時，計算“表達式1”的值。

  1. 如果“表達式1”的值為真，則執行“語句1”。執行完“語句1”，則跳過整個if-elif-else語句。
  2. 如果“表達式1”的值為假，則計算“表達式2”的值。如果“表達式2”的值為真，則執行“語句2”。執行完“語句1”，則跳過整個if-elif-else語句。
  3. 如果“表達式2”的值為假，則計算“表達式3”的值。
  4. …
  5. 直到某個表達式的值為真才停止
  6. 如果沒有一個真的表達式且有else，則執行“語句x”。如果沒有一個真的表達式且沒有else，則跳過整個if-elif-else語句。
  graph TD
A[計算表達式1的值] 
A --> C[表達式1的值為真]
A --> D[表達式1的值為假]
D --> E[計算表達式2的值]
C --> F[執行語句1]
F --> G[跳出整個if-else語句]
E --> H[表達式2的值為真]
E --> I[表達式2的值為假]
H --> J[執行語句2]
J --> K[跳出整個if-else語句]
I --> L[...]
L --> M[沒有表達式是真的]
M --> N[有else]
M --> o[沒有else]
N --> p[執行語句x]
o --> q[跳出整個if-else語句]
  • 每個el都是對它上面所有表達式的否定
