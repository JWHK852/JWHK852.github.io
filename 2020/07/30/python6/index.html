Python6-運算符与表達式

發表於2020-07-30|更新於2020-08-01|計算機語言

表達式

由變量、常量和運算符組成的式稱之為表達式

運算符和表達式

算術運算符和算術運算表達式

算術運算符

  • 加：+
  • 減：-
  • 乘：*
  • 除：/
  • 取模（取餘數）：%
  • 求冪（次方）：**
  • 取整（商）：//

算術運算表達式

例子：

  • 1 + 1
  • 2 * 3

功能

進行相關符號的數學運算（不會改變變量的值）

值

相關的數學運算結果

賦值運算符和賦值運算表達式

賦值運算符

基本的賦值運算符是“=”

賦值運算表達式

格式：變量 = 表達式
功能：計算等號右側“表達式”的值，並賦值給等號左側的變量
值：賦值結束後變量的值

復合運算符

  • +=
    • a += b → a = a + b
  • -=
    • a -= b → a = a - b
  • *=
    • a *= b → a = a * b
  • /=
    • a /= b → a = a / b
  • %=
    • a %= b → a = a % b
  • **=
    • a **= b → a = a ** b
  • //=
    • a //= b → a = a // b

關係運算符和關係運算符表達式

關係運算符

  • 等於： ==
  • 不等於： !=
  • 大於： >
  • 小於： <
  • 大於或等於： >=
  • 小於或等於： <=

關係運算符表達式

格式

功能

計算“表達式1”和“表達式2”的值

值

如果關係成立，整個關係運算表達式為“真(True)”
如果關係成立，整個關係運算表達式為“假(False)”

邏輯運算符

邏輯與

邏輯與運算符： and

邏輯與運算表達式

值：

  • 如果“表達式1”和“表達式2”的值為“真”，整體邏輯與運算表達式為“真”
  • 如果“表達式1”和“表達式2”任意一個的值為“假”，整體邏輯與運算表達式為“假”
  • 如果“表達式1”和“表達式2”的值為“假”，整體邏輯與運算表達式為“假”

邏輯或

邏輯或運算符： or

邏輯或運算表達式

值：

  • 如果“表達式1”和“表達式2”的值為“真”，整體邏輯或運算表達式為“真”
  • 如果“表達式1”和“表達式2”任意一個的值為“真”，整體邏輯或運算表達式為“真”
  • 如果“表達式1”和“表達式2”的值為“假”，整體邏輯或運算表達式為“假”

邏輯非

邏輯非運算符： not

邏輯非運算表達式

值：

  • 如果“表達式”的值為“真”，整體邏輯非運算表達式為“假”
  • 如果“表達式”的值為“假”，整體邏輯非運算表達式為“真”

成員運算符

in

定義

如果在指定的序列中找到值，返回True，否則返回False

not in

定義

如果在指定的序列中沒有找到值，返回True，否則返回False

身份運算符

is

判斷兩個標識符是不是引用同一個對象

is not

判斷兩個標識符是不是引用不同的對象

位運算符

定義

按位運算符是把數字看做二進制數進行計算

符號

按位與運算符：&

相應的位數都為1，則該位的結果為1，否則為0

按位或運算符：|

兩個二進制位有一位1時，該位的結果為1

按位異或運算符：^

二進制的兩位相異時，結果為1

按位取反運算符：~

每個二進制數據位取反,1變0，0變1

左移動運算符：<<

各二進制位全部左移動若干位，由<<右側的數字決定，高位丟棄，低位補0

右移動運算符：>>

各二進制位全部右移動若干位，由>>右側的數字決定，低位丟棄，高位補0

運算符優先級

排名分先後

  • 最高
    • **
  • 第二
    • ~
    • +（正號）
    • -（負號）
  • 第三
    • /
    • %
    • //
  • 第四
    • +（加）
    • -（減）
  • 第五
    • >>
    • <<
  • 第六
    • &
  • 第七
    • ^
    • |
  • 第八
    • <=
    • >=
    • >
    • <
  • 第九
    • ==
    • !=
  • 第十
    • =
    • %=
    • +=
    • -=
    • *=
    • /=
    • **=
  • 第十一
    • is
    • is not
  • 第十二
    • in
      +not in
  • 第十三
    • not
    • or
    • and

短路原則

and：

如果“表達式1”為“False”，則直接跳過後續表達式計算

or

如果“表達式1”為“True”，則直接跳過後續表達式計算
