Django3-視圖

發表於2022-12-23|更新於2023-06-23|計算機語言
|閱讀量:

視圖

  • 作用
    • 接受web請求，並且響應web請求
  • 本質
    • 視圖為一個python函數
  • 響應

    • 網頁

      • 重定向
      • 錯誤視圖
        • 404
        • 500
        • 400

    • Json數據

  • 響應過程
  graph TD
    A["瀏覽器輸入網址\nwww.dvagry.com/dvagary/index.html"] -->|"網址"|B["Django獲取網址信息\n去除IP與端口\ndvagary/index.html"]
    B-->|"虛擬路徑與文件名"|C["URL管理器\n逐個匹配urlconf\n記錄視圖函數名"]
    C-->|"視圖函數名"|D["視圖管理器\n找到對應的視圖去執行\n返回結果給瀏覽器"]
    D-->|"響應得數據"|A

URL配置

  • 配置流程
    • 指定根級url配置文件
      • settings.py中的ROOT_URLCONF
      • ROOT_URLCONF = 'project.urls'
      • 已經默認實現
    • urlpatterns
      • 一個url實例列表
      • URL對象
        • 正則表達式
        • 視圖名稱
        • 名稱
    • url匹配正則的注意事項
      • 如果想要從url獲取一個值，需要對正則表達式加上括號()
      • 匹配正則前方無需加入/
      • www.abc.com/abc/def
        • ^abc/def$
      • 正則表達式前需要加r

引入其他url配置

  • 一般在應用中創建urls.py文件，定義本應用的url配置，在過程urls.py文件中使用include()方法

URL反向解析

  • 概念
    • 如果在視圖或者模板中使用了硬編碼的鏈接，在url配置發生改變時，動態生成鏈接的地址
  • 解決
    • 在使用鏈接時，通過url配置的名稱，動態生成url地址
  • 作用
    • 使用url模板

視圖函數

  • 定義視圖
    • 本質
      • 函數
    • 視圖參數
      • 一個HTTP Request的實例
      • 通過正則表達式獲取的參數
    • 位置
      • 一般在views.py文件下定義
  • 錯誤視圖

    • 404 視圖

      • 找不到網頁(url匹配不成功)時返回
      • 在templates目錄下定義404.html
        • request_path
          • 導致錯誤的網址
      • 配置settings.py
        • ALLOWED_HOSTS = ['*']
        • DEBUG
          • 如果為True，永遠不會調用錯誤頁面

    • 500 視圖

      • 在視圖代碼中出現錯誤（服務器代碼）

    • 400 視圖

      • 錯誤出現在客戶操作

HttpRequest對象

概述

  • 伺服器接受http請求後，根據報文創建HttpRequest對象
  • 視圖第一個參數為HttpRequest對象
  • Django創建的，之後調用視圖時傳遞給視圖

屬性

  • path
    • 請求的完整路徑(不包括域名，端口)
  • method
    • 請求的方式，常有的有GET和POST
  • encoding
    • 表示瀏覽器提交的數據編碼方式
      • 一般為UTF-8
  • GET
    • 類似字典的對象，包含了GET請求的所有參數
  • POST
    • 類似字典的對象，包含了POST請求的所有參數
  • FILES
    • 類似字典的對象，包含了所有上傳的文件
  • COOKIES
    • 字典，包含所有的COOKIES
  • session
    • 類似字典的對象，表示當前的會話

方法

  • is_ajax()
    • 如果是通過XMLHttpRequest發起的，返回True

QueryDict對象

  • request對象中的GET,POST都屬於QueryDict對象

方法

  • get()
    • 根據鍵獲取值
    • 只能一個
    • www.dvagary.com/abc?a=1&b=2&c=3
  • getlist()
    • 將鍵的值以列表返回
    • 獲取多個值
    • www.dvagary.com/abc?a=1&a=2&c=3

GET屬性

  • 獲取瀏覽器傳遞給伺服器的數據
  • www.dvagary.com/abc?a=1&b=2&c=3
  • www.dvagary.com/abc?a=1&a=2&c=3

POST屬性

  • 使用表單提交實現POST請求
