AI3-機器學習-線性回歸

發表於2022-05-13|更新於2022-05-25|計算機語言
|閱讀量:

機器學習

  • 一種實現人工智能的方法
    • 從數據中尋找規律，建立關係，根據建立的關係解決問題

應用場景

  • 數據挖掘
  • 計算機視覺
  • 自然語言處理
  • 證券分析
  • 醫學診斷
  • 機器人
  • DNA測序

類別

  • 監督學習 Supervised Learning
    • 訓練數據包括正確結果
      • 線性回歸
      • 邏輯回歸
      • 決策樹
      • 神經網絡，卷積神經網絡，循環神經網絡
  • 無監督學習 Unsupervised Learning
    • 訓練數據不包括正確結果
      • 聚類算法
  • 半監督學習 Semi-supervised Learning
    • 訓練數據包括少量正確結果
  • 強化學習 Reinforcement Learning
    • 根據每次結果的反饋學習實現優化
  • 混合學習
    • 監督學習 + 無監督學習

回歸分析

  • 回歸分析 Regression Analysis
    • 根據數據，確定兩種或者兩種以上變量間互相依賴的定量關係
    • 函數表達式
      • y = f(x1,x2,…,xn)
    • 例子
      • 年齡預測身高
      • 住宅面積預測房價
  graph LR
    A[回歸] -->B(變數量)
    A --> C(函數關係)
    B -->D("一元回歸：y=f(x)")
    B -->E("多元回歸：y=f(x1,x2,...,xn)")
    C --> F("線性回歸：y=ax+b")
    C-->G("線性回歸：y=ax²+bx+c")

線性回歸

  • 變量與因變量存在線性關係
    • 例子
      • 線性回歸
        • 距離=速度x時間
      • 非線性回歸
        • 距離=加速度x時間²

回歸問題求解

  • 尋找最適合的a,b(y=ax+b)
    • 假設x為變量，y為結果，y’為模型輸出結果，目標為y’盡可能接近y
    • 損失函數J降到最小
    • 例子

      J1=12×3×((0.5−1)2+(1−2)2+(1.5−3)2)=0.583J_1=\frac1{2\times3}\times\left(\left(0.5-1\right)^2+\left(1-2\right)^2+\left(1.5-3\right)^2\right)=0.583J1​=2×31​×((0.5−1)2+(1−2)2+(1.5−3)2)=0.583
      J2=12×3×((4−1)2+(3−2)2+(2−3)2)=1.83J_2=\frac1{2\times3}\times\left(\left(4-1\right)^2+\left(3-2\right)^2+\left(2-3\right)^2\right)=1.83J2​=2×31​×((4−1)2+(3−2)2+(2−3)2)=1.83
    • 可見y’₂更接近y

損失函數

  • 評估模型與真實數據差距

梯度下降法

  • 尋找最小值的一種方法，通過向函數上當前點對應的梯度（或近似的梯度）的反方向的規定步長距離點進行迭代搜索，直到在極小點收斂
  • J=f(p)J=f\left(p\right)J=f(p) 搜索方法 pi+1=pi−α∂∂pif(pi)p_{i+1}=p_i-\alpha\frac{\partial{}}{\partial p_i}f\left(p_i\right)pi+1​=pi​−α∂pi​∂​f(pi​)
  • pᵢ為初始值，α為步長
  • 缺點
    • 初始值的選擇可能導致梯度下降法計算到局部最小值
  • 例子
    • 設J=f(p)=3.5p2−14p+14J=f\left(p\right)=3.5p^2-14p+14J=f(p)=3.5p2−14p+14, pᵢ=0.5, α=0.01
      ∂∂pif(pi)=−10.5\frac{\partial{}}{\partial p_i}f\left(p_i\right)=-10.5∂pi​∂​f(pi​)=−10.5
      pi+1=0.5−(0.01)(−10.5)=0.605p_{i+1}=0.5-(0.01)(-10.5)=0.605pi+1​=0.5−(0.01)(−10.5)=0.605
    • 可見逐漸接近最小點x=2
    • 不斷進行迭代可以無限接近最小點

Scikit-learn
  • 文檔
  • Python中針對機器學習的開源框架
  • 實現數據預處理，分類，回歸，降維，模型選擇等常用機器學習算法
  • 特點
    • 集成機器學習中各類成熟算法
    • 不支持Python以外的語言
    • 不支持深度學習和強化學習

評估模型表現
  • y 與 y’ 的均方誤差(MSE)
    • MSE越小誤差越小
  • R方值(R²)
    • R²越接近1誤差越小
