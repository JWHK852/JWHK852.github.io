Solidity Contract

數據類型

mapping

  • 用於映射值
  • mapping(_KeyType => _ValueType)
  • e.g：以人名查詢該人喜愛的數字

陣列Array

  • 類型[] Array名稱
  • e.g：People類型的名為peopleList的Array，可見度為public

添加數據

  • 使用push()為Array添加數據
  • e.g：將John添加至peopleList內

FundMe Contract

Constructor

  • 初始化Contract
  • e.g：設置owner為該Contract的部署者

payable

  • 設置該功能為支付功能
  • e.g：支付

查詢實時ETH兌換USD

  • 導入AggregatorV3Interface.sol
  • import "@chainlink/contracts/src/v0.8/interfaces/AggregatorV3Interface.sol";
  • Ethereum Data Feeds 查詢網站
  • 建立getPrice()功能
  • call getPrice() 後得到值，除10^8為1ETH兌換美金值
    • 例如得到值為120346702341，那1ETH可兌換1203.46702341USD

Modifier

  • 用於定義一個功能的行為，意思為在運行該功能前先運行Modifier內的代碼
  • e.g：使用require判斷msg的發送者是否為合約部署者
  • 使用方法
    • 在功能聲明
    • e.g：取得款項功能withdraw
